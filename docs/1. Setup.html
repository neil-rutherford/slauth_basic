Welcome and Set-up

Welcome

Welcome to Slauth Basic, a boilerplate login module that allows you to start Flask projects quickly. Included in this package:

  • Basic login functionality. This module assumes that all users have equal permissions. It includes basic register-login-logout functionality.
  • Two pages. One page requires a login and displays user-specific information. The other page does not require a login and displays a generic greeting.
  • Password reset. This module sends asynchronous emails to allow users to reset their passwords.
  • Email verification and email opt-in. This module verifies a user's email address and records their email preferences, which is useful for email marketing.
  • Error handling. This module contains an error handler, which displays customizable error messages for 404s and 500s.

We hope this makes your Flask project run smoothly. In this section, we talk about how to set up the app. In the next section, we talk about how to use and troubleshoot the app. In the third section, we discuss how to add on to the app. In the final section, we provide contact and licensing information.


Set-up

Getting your environment ready

If you do not already have Python 3.x installed, you may find instructions for how to do so here.

Open a terminal and navigate to the slauth_basic directory. In your terminal, run the following code:

  1. python -m venv venv
  2. For Windows: venv\Scripts\activate || For Unix/Linux: source venv/bin/activate
  3. python -m pip install -r requirements.txt
  4. flask db init
  5. flask db migrate -m "Initial commit"
  6. flask db upgrade

By running this code, you have set up a virtual environment, downloaded all dependencies, and initialized your database.

Setting your configuration variables

In our next step, we will set configuration variables that allow the application to run smoothly. These instructions assume you are running this on your local machine. In your terminal, run the following code:

  1. export SECRET_KEY={fill in your secret key here}
    (The secret key is used for encryption and can be thought of as your website's password.)
  2. export SQLALCHEMY_DATABASE_URI={fill in your database uri here}
    (This defaults to SQLite. If you are okay using SQLite, skip this step. If you want to use a SQL server like MySQL or Postgres, enter the database URI here.)
  3. export MAIL_SERVER={fill in your mail server here}
    (Look up your email provider's information. If you are using Gmail, for example, this would be "smtp.gmail.com".)
  4. export MAIL_PORT={fill in your mail port here}
    (Look up your email provider's information. If you are using Gmail, for example, this would be 465 if you want SSL and 587 if you want TLS.)
  5. export MAIL_USE_TLS={fill in your boolean preference here}
    (If you want to use TLS, set this to True. Otherwise, set it to False.)
  6. export MAIL_USERNAME={fill in your email address here}
    (What is your email username? Example: "iloveslauth@gmail.com")
  7. export MAIL_PASSWORD={fill in your password here}
    (What is your email password? Example: "password123")

If you want to disable email functionality, leave all of the "MAIL_" variables blank.

Running the app

Running the app is as easy as entering flask run into your terminal. Your web app will run on localhost (http://127.0.0.1:5000/). To stop the app, go to the terminal and press Ctrl + C.
