Using the app

Endpoints

/ and /home

This page is not login restricted. It displays the lorem ipsum text as a placeholder.

/dashboard

This page is login restricted. It displays a welcome message and shows the user all of the information we have about them in the database.

/register

This is where new users go to register. After filling out the form, they are saved in the database as a user and are logged in.

/login

This is how existing users log in to the platform. If successful, they will be redirected to their dashboard. If unsuccessful, they will be prompted for their credentials again. Please note that Slauth Basic offers no brute-force protection.

/logout

This is how users log out. They are redirected to the home page.

/reset-password-request

This is how users reset their password. Entering their email address will cause an email to be sent to their email address. If email functionality is disabled, this will result in a 500 error.
